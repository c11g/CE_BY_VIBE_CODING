# MyBoard 기능

- 텍스트 드래그 시 컬러 선택 툴팁 표시
- 8가지 파스텔톤 컬러 버튼 (빨강, 주황, 노랑, 연두, 파랑, 보라, 갈색, 검정)
- 선택된 텍스트에 하이라이트 적용 (투명도 60%)
- 하이라이트 자동 넘버링 (빨간 원형 배지, 1부터 시작, 투명도 80%)
- 하이라이트된 텍스트 클릭 시 개별 제거 툴팁 표시
- 모든 하이라이트 한번에 제거 기능 (넘버링 카운터 리셋)
- 📷 영역 캡쳐 기능 (드래그로 원하는 부분만 캡쳐 후 다운로드)ome Extension

바닐라 JavaScript로 개발된 크롬 익스텐션입니다.
웹페이지에서 텍스트를 드래그하면 툴팁이 나타나는 기능을 제공합니다.

## 기능

- 텍스트 드래그 시 컬러 선택 툴팁 표시
- 8가지 파스텔톤 컬러 버튼 (빨강, 주황, 노랑, 연두, 파랑, 보라, 갈색, 검정)
- 선택된 텍스트에 하이라이트 적용 (투명도 60%)
- 하이라이트 자동 넘버링 (빨간 원형 배지, 1부터 시작)
- 하이라이트된 텍스트 클릭 시 개별 제거 툴팁 표시
- 모든 하이라이트 한번에 제거 기능 (넘버링 카운터 리셋)

## 개발 환경

- Vanilla JavaScript
- Chrome Extension Manifest V3

## 파일 구조

```
├── manifest.json       # 익스텐션 설정 파일
├── background.js       # 백그라운드 서비스 워커
├── content.js          # 콘텐츠 스크립트 (메인 기능)
├── icons/              # 아이콘 파일들
└── package.json        # 프로젝트 설정
```

## 사용 방법

### 하이라이트 추가
1. 웹페이지에서 텍스트 드래그
2. 컬러 선택 툴팁이 나타남
3. 원하는 파스텔톤 컬러 버튼 클릭하여 하이라이트 적용

### 하이라이트 제거
4. 🗑️ "ALL 🗑️" 버튼으로 페이지의 모든 하이라이트 제거
5. 하이라이트된 텍스트를 클릭하여 개별 하이라이트 제거 툴팁 표시

### 영역 캡쳐
6. 📷 "영역 캡쳐" 버튼 클릭
7. 마우스로 드래그하여 캡쳐할 영역 선택
8. 선택한 영역이 PNG 파일로 자동 다운로드

## 개발 방법

1. 크롬에서 `chrome://extensions/` 접속
2. "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택
