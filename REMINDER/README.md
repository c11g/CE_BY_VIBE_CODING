# Reminder

## 프로젝트 개요

- Chrome 확장 프로그램(Extension)으로, 브라우저 사용 중 빠르게 리마인더를 등록하고 관리하기 위한 도구입니다.
- 가벼운 개인 생산성 도구를 목표로 하며, 복잡한 일정 관리보다는 "지금 보고 있는 것과 관련된 할 일"을 빠르게 적어두고 잊지 않도록 해주는 데 초점을 둡니다.

## 목표

- **빠른 입력**: 툴바 아이콘 클릭 또는 단축키로 즉시 리마인더를 추가할 수 있어야 합니다.
- **간단한 관리**: 등록된 리마인더를 리스트 형태로 한눈에 보고, 완료/삭제를 쉽게 할 수 있어야 합니다.
- **브라우저 환경에 최적화**: 로그인/동기화 여부와 무관하게, 최소한 현재 브라우저 프로필 내에서는 안정적으로 데이터가 유지되어야 합니다.
- **방해 최소화**: 알림은 필요할 때만, 과하지 않게 노출되도록 설계합니다.

## 기본 기능 정의 (v1)

- 브라우저 툴바 아이콘을 클릭하면 팝업이 열린다.
- 팝업에서 **제목 + 시간**을 입력하고 `저장`하면 해당 시각에 알람이 예약된다.
- 예약된 시각이 되면 **Chrome 알림(Notification)** 과 함께 **알람 사운드**가 재생된다.
- 팝업 상단의 `알람 끄기` 버튼으로, 재생 중인 알람을 한 번에 멈출 수 있다.
- 팝업 하단 `Upcoming` 영역에서 **앞으로 울릴 알람의 목록(시간 + 제목)** 을 확인할 수 있다.

## 앞으로의 확장 아이디어

- 리마인더에 대한 편집 및 삭제
- 리마인더 데이터 백업/복원 (동기화 스토리지 또는 파일 내보내기)
