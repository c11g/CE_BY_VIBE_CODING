# CE BY VIBE CODING

이 프로젝트는 바이브 코딩 학습을 목적으로 제작한 간단한 Chrome 익스텐션들을 관리하는 모노레포입니다.

## 익스텐션 목록

### MY_BOARD
- **설명**: 게시판 관리 Chrome 익스텐션
- **위치**: `/MY_BOARD/`
- **상태**: 개발 중

### REMINDER
- **설명**: 브라우저 사용 중 빠르게 리마인더를 등록하고 관리하는 도구
- **위치**: `/REMINDER/`
- **상태**: 개발 중

## 개발 환경 설정

각 익스텐션은 독립적으로 개발되며, 각 폴더 내의 README.md를 참조하세요.
